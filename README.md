 # Setup
 - Download the Zip file or Clone the repo with:
   - git clone https://github.com/karam084/blog-app-rails
   - To access the project directory
      - cd blog-app-rails

 - Install the app
    - bundle install
    - rails db:migrate

 - Run the App
    - rails s 
    - Check http://localhost:3000/ in your browser
